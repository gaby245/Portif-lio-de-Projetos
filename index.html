Meu jogo: O Cristal Perdido

O projeto é um jogo interativo baseado em uma página web, desenvolvido com HTML, CSS e JavaScript. O objetivo é criar uma experiência de narrativa onde o jogador avança por etapas de uma história ou missão ao clicar no botão "Próximo". A cada nova etapa, o jogador é apresentado a novos desafios, informações ou escolhas que o levam a descobrir mais sobre o enredo.

A estrutura do site utiliza HTML para organizar o conteúdo semântico, como as divisões de etapas da história e a navegação entre elas. O CSS é responsável por estilizar a página, criando uma atmosfera visual envolvente, com cores e tipografia que ajudam a imergir o jogador na experiência do jogo. Além disso, o JavaScript é usado para controlar a interação entre o jogador e a história, permitindo que as etapas se alternem conforme o clique no botão de avanço.

O HTML organiza a página com um cabeçalho que introduz o conceito do jogo e uma área principal (.container) onde a história se desenrola em etapas (.passo). Cada "passo" é um bloco de conteúdo que se torna visível à medida que o jogador avança, e o design é simplificado para manter o foco na narrativa e na interatividade. A transição entre os "passos" é controlada por JavaScript, o que permite que o jogador tenha controle sobre o ritmo da história.

O CSS personaliza o visual do jogo, utilizando cores e fontes que criam um ambiente imersivo, com um design simples e eficiente. A utilização de propriedades de layout, como flexbox, permite que o conteúdo se ajuste de forma fluida a diferentes tamanhos de tela. O projeto foca em uma experiência de usuário intuitiva, com uma interface simples, mas envolvente, permitindo que o jogador se concentre na narrativa sem distrações desnecessárias.

Este projeto é um exemplo de como é possível criar uma experiência de jogo envolvente com tecnologias web básicas, utilizando HTML, CSS e JavaScript de maneira simples e eficaz. O objetivo é criar uma história interativa que pode ser expandida com novas etapas e funcionalidades, como a introdução de escolhas ou a personalização da narrativa.

Meu site: GABYMINAMFLIX

O projeto GABYMINAMFLIX é um site dedicado ao Studio Ghibli, um dos estúdios de animação mais renomados mundialmente, com um design moderno e sofisticado. O objetivo principal é fornecer uma plataforma para os fãs do Studio Ghibli, onde podem explorar e descobrir filmes do estúdio, com uma navegação visualmente agradável e interativa.

A estrutura do site é construída com HTML, que organiza a semântica e o conteúdo, e CSS, que estiliza a página para proporcionar uma experiência agradável ao usuário. O design do site é baseado em um esquema de cores verdes suaves, combinados com textos brancos, criando um contraste visual que torna a navegação fluída e confortável. O layout é responsivo, garantindo que a experiência do usuário seja otimizada em dispositivos de diferentes tamanhos de tela, como desktops e dispositivos móveis.

O HTML é usado para estruturar o conteúdo da página, incluindo uma lista de filmes destacados do Studio Ghibli. Cada filme é apresentado com uma imagem de capa que ganha destaque visual ao passar o mouse sobre ela, utilizando efeitos de hover que tornam a interação mais interessante. A navegação é facilitada por meio de botões e links interativos, que levam o usuário a seções adicionais do site, como descrições dos filmes e vídeos relacionados.

O CSS, por sua vez, é responsável por criar a estética visual do site, utilizando propriedades de layout flexível para garantir que os elementos se ajustem adequadamente a qualquer tamanho de tela. As imagens dos filmes são tratadas com efeitos visuais, como transições suaves e hover effects, para melhorar a interação. Além disso, o design utiliza fontes importadas do Google Fonts para garantir uma tipografia agradável e legível.

O JavaScript é utilizado para controlar a interatividade do site, principalmente no que diz respeito aos efeitos de hover nas imagens dos filmes e nas animações de transição entre as seções. O código é simples e eficiente, garantindo uma experiência de navegação sem interrupções.

Este projeto é um exemplo de como tecnologias web básicas podem ser usadas para criar uma plataforma interativa e visualmente atraente, sem a necessidade de scripts complexos. O foco é proporcionar uma experiência agradável aos usuários, oferecendo uma navegação intuitiva e acessível, ideal para fãs do Studio Ghibli que desejam explorar os filmes do estúdio de maneira simples e envolvente.

Meu jogo: Trilha da joaninha

O projeto Trilha da Joaninha é um jogo interativo simples, desenvolvido utilizando o Scratch, que proporciona uma experiência divertida e educativa. No jogo, o objetivo é guiar a joaninha para seguir o ponteiro do mouse enquanto ela deixa uma trilha preta pelo caminho. A cada movimento, a joaninha cria um padrão visual interessante, e o jogo continua até que o usuário clique em qualquer parte da tela, interrompendo a ação. Esse tipo de jogo simples é ideal para explorar conceitos de movimento e animação de maneira intuitiva e envolvente.

A estrutura do projeto é construída com o Scratch, que utiliza blocos de código visuais para programar a interação entre a joaninha e o usuário. O movimento da joaninha é controlado por um conjunto de blocos que a fazem seguir o ponteiro do mouse. Isso cria uma interação direta entre o jogador e o jogo, tornando-o imersivo e interessante. A trilha deixada pela joaninha é gerada por blocos de código que desenham uma linha preta sempre que a joaninha se move.

O jogo é focado na simplicidade e na diversão, sem a complexidade de muitos jogos mais avançados. A ideia é oferecer uma experiência acessível para iniciantes, ao mesmo tempo que explora conceitos importantes de programação, como movimento, controle de eventos e animação. O jogo pode ser uma ferramenta educativa, ideal para ensinar os fundamentos da programação a jovens aprendizes, enquanto eles se divertem com a interatividade.

Embora o jogo tenha uma mecânica simples, ele permite que o jogador explore diferentes caminhos e padrões à medida que move a joaninha pela tela. Isso cria um ambiente descontraído e livre de pressões, ideal para quem está começando a entender como a programação pode ser aplicada para criar jogos. O design visual do jogo é minimalista, focado em um único elemento (a joaninha) e sua interação com o ambiente ao seu redor, proporcionando uma experiência visualmente limpa e agradável.

Este projeto é uma ótima demonstração de como é possível criar jogos e experiências interativas simples utilizando o Scratch. Ele foca na interação básica e no movimento, utilizando as ferramentas do Scratch de maneira criativa para criar um jogo divertido e educativo que pode ser expandido ou modificado para criar novas funcionalidades ou desafios para o jogador.
